READ ME
----------------------------------------------------------------------------

1.	Download and add JADE-libraries "jade.jar" and "commons-codec-1.3".
2.	Build and compile the project.
3.	Compile project with "jade.Boot" as the main-class.
4.	Use argument "-gui" before execution.
5.	Run the program.
6.	Start agents from the classes "ProfilerAgent", "TourGuideAgent" and "CuratorAgent".
